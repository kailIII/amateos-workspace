Marketscraper
Marketscraper es un proyecto desarrollado por Alberto Mateos para la asignatura AAP de la ETSIIT de la UGR.

Objetivo

El objetivo del proyecto marketscraper es extraer datos acerca de algunos de los productos que se venden en algunos de los supermercados más importantes del país. En concreto, la extracción de datos se va a realizar a partir de la web carritus.com De esta forma, una vez extraídos dichos datos, éstos son almacenados en una base de datos que, por exigencias de la asignatura, estará basada en XML (eXist DB). Posteriormente, el usuario podrá recuperar los datos almacenados en dicha base de datos a través de esta página web, seleccionando los tipos de productos a mostrar a partir del menú o bien utilizando el buscador.
